# 🚀 Portfolio Personnel - Aymane Dahani

> Mini-projet WEB réalisé dans le cadre du module "Programmation WEB" à la FSSM (Faculté des Sciences Semlalia Marrakech).

## 📖 À propos du projet

Ce projet est un **portfolio personnel** statique développé avec HTML5, CSS3 et JavaScript vanilla. Il présente mes compétences, mes projets et mes coordonnées de manière moderne et interactive.

## 🌐 Démo en ligne

🔗 **[Voir le site en direct](https://dahaniaymane052-dot.github.io/portfolio-website/)**

## 🛠️ Technologies utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Flexbox, Grid, Media Queries, Animations
- **JavaScript (Vanilla)** - Interactivité et manipulation du DOM

> ⚠️ Aucun framework utilisé (ni Bootstrap, ni jQuery)

## ✨ Fonctionnalités

- 🎨 Design moderne et épuré
- 🌙 Mode sombre / clair (Dark/Light mode)
- 📱 Entièrement responsive (mobile, tablette, desktop)
- 🍔 Menu burger pour mobile
- ⚡ Animations CSS fluides
- 📊 Barres de compétences animées
- 🎯 Filtres dynamiques pour les projets
- ✉️ Formulaire de contact avec validation JavaScript 


## 📋 Pages du site

| Page | Description |
|------|-------------|
| 🏠 **Accueil** | Hero section, présentation, services |
| 👤 **À propos** | Bio détaillée, parcours, timeline |
| 💼 **Compétences** | Barres animées, outils maîtrisés |
| 📁 **Projets** | Galerie filtrable de mes réalisations |
| ✉️ **Contact** | Formulaire avec validation, réseaux sociaux |
